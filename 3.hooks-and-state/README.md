# ⚛️ React Hooks and State

This project is created to practice React State and Hooks and understand state management, component lifecycle, form handling, and API integration through simple examples.

---

## 📚 Concepts Covered

### 🔹 Counter.jsx
- Demonstrates `useState`.
- Increases, decreases, and resets count values.
- Understands state updates and component re-rendering.

### 🔹 LifeCycle.jsx
- Demonstrates `useEffect`.
- Understands component mount, update, and unmount.
- Uses cleanup functions.

### 🔹 FormWithState.jsx
- Demonstrates multiple `useState` hooks.
- Handles form inputs.
- Manages multiple pieces of state.

### 🔹 FormWithObject.jsx
- Demonstrates object state management.
- Updates object properties using the spread operator.
- Handles related data in a single state object.

### 🔹 FetchUsers.jsx
- Demonstrates API calls using Fetch API.
- Stores API data in state.
- Displays dynamic data using `map()`.

### 🔹 AxiosUsers.jsx
- Demonstrates API calls using Axios.
- Handles API responses.
- Renders fetched data dynamically.

## 📝 React Definitions

### State
State is data stored inside a React component. When the state changes, the component automatically re-renders.

### useState
`useState` is a React Hook used to create and manage state in functional components.

### useEffect
`useEffect` is a React Hook used to perform side effects such as API calls, timers, subscriptions, and lifecycle operations.

### Component Mount
The process when a component is created and displayed on the screen for the first time.

### Component Update
The process when a component re-renders because its state or props have changed.

### Component Unmount
The process when a component is removed from the screen and cleanup operations are performed.

### Fetch API
A built-in JavaScript API used to make HTTP requests and retrieve data from a server.

### Axios
A third-party library used to make HTTP requests with simpler syntax and automatic JSON data handling.

### map()
A JavaScript array method used to iterate over an array and render JSX elements dynamically.
