## вњЁ Styling Methods Covered

### рџ”№ Inline CSS

* Uses the `style` prop.
* Styles are written directly inside JSX.
* Best for small or dynamic styles.

### рџ”№ External CSS

* Uses a separate `.css` file.
* Styles are applied using `className`.
* Suitable for reusable and global styles.

### рџ”№ CSS Modules

* Uses `.module.css` files.
* Provides locally scoped styles.
* Prevents CSS class name conflicts.

### рџ”№ React Bootstrap

* Uses pre-built Bootstrap components.
* Creates responsive and modern forms quickly.
* Reduces the need for custom CSS.
