# 📚 Array & Object Methods Revision

This folder contains JavaScript array and object methods that are frequently used in React applications.

---

## Files

### Destructure.jsx
Purpose:
Extract values from objects or arrays into variables.

### Map.jsx
Purpose:
Transform data and return a new array.

### Filter.jsx
Purpose:
Return elements that satisfy a condition.

### Reduce.jsx
Purpose:
Convert an array into a single value.

### Rest.jsx
Purpose:
Collect multiple arguments into one array.

### Spreading.jsx
Purpose:
Expand arrays or objects.

---

# Interview Quick Notes

Destructuring ➜ Extract values
map()         ➜ Transform data
filter()      ➜ Select data
reduce()      ➜ Single value
rest(...)     ➜ Collect values
spread(...)   ➜ Expand values
